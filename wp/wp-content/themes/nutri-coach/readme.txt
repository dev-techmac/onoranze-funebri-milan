== Description ==

Nutri Coach is a modern, fully customizable WordPress block theme designed for nutrition coaches, health consultants, and wellness professionals. Built with Full Site Editing capabilities and fully compatible with the AllCoach plugin, this theme makes it easy to build a professional coaching website without any coding knowledge.

Perfect for:
* Nutrition Coaches & Dietitians
* Health & Wellness Consultants
* Personal Trainers with Nutrition Focus
* Functional Medicine Practitioners
* Online Coaching Programs
* Meal Planning & Diet Services

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
